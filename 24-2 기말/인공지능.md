### 캡슐화
- 클래스의 속성을 외부에서 직접 접근하는 것을 제한
- 고의 또는 실수에 의해 값이 변경되어 오류가 발생하는 것을 방지(데이터 보호)
```python
class Car:
	def setSpeed(self, speed) :
		if(speed < 0) :
			self.__speed = 0 # 멤버 변수 앞 '__'를 붙여 사용
		elif(speed > 400) :
			self.__speed = 400
		else :
			self.__speed = speed
	def getSpeed(self) :
		return self.__speed
```

### 생성자
- 인스턴스가 생성될 때 자동으로 호출되는 메서드
- 인스턴스에 초기값을 설정해야 할 필요가 있을 때 생성자를 구현하는 것이 안전한 방법
- 생성자에 매게변수가 있는 경우 인스턴스 생성시 해당하는 값을 전달해야 함
```python
class Car:
	# 생성자 정의
	def __init__(self, speed, color) :
		self.setSpeed(speed)
		self.setColor(color)

car1 = Car(100, '파랑')
print(car1.getSpeed()) # 100
print(car1.getColor()) # 파랑
```

### 소멸자
- 인스턴스가 소멸될 때 자동으로 호출되는 메서드
- 어떤 인스턴스가 없어지기 전에 뭔가 처리를 필요로 한다면 소멸자가 유용
```python
class Car:
	# 소멸자 정의
	def __del__(self) :
		print('폐차')

car1 = Car()
del car1 # '폐차' 출력
```
