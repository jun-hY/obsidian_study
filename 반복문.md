
### 반복문의 문법
for, while, do/while

while
```c
#include <stdio.h>

int main()
{
    int a = 90;
    while (a < 101)
    {
        printf("%2d ", a++);
    }
    printf("\n");
    return 0;
}
```

do / while
```c
#include <stdio.h>

int main()
{
    int a = 0;
    do
    {
        printf("%2d", a++);
    } while (a > 5);
    printf("\n");
    return 0;
}
```
